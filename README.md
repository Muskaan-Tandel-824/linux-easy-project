# Linux Easy Project – File & Permission Management

## Project Overview
This project demonstrates basic Linux administration skills including directory creation, file management, and permission handling.

## Directory Structure
project/
├── docs
│   └── readme.txt
├── scripts
│   └── run.sh
└── logs
    └── app.log

## Tasks Performed
- Created directory hierarchy using mkdir
- Created files using touch
- Assigned permissions using chmod
- Verified permissions using ls -l

## Permissions Used
- readme.txt → 644
- run.sh → 755
- app.log → 600

## Commands Used
mkdir, touch, chmod, ls -l

## Platform
Linux (Amazon Linux / Ubuntu)

## Outcome
Learned Linux file system management and access control.
